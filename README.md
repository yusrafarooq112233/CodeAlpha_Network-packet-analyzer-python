# Network Packet Analyzer Using Python (Scapy)

## Overview

This project implements a basic Network Packet Analyzer using Python and the Scapy library. It captures live network packets and extracts useful information such as source and destination IP addresses, protocol type, port numbers, packet size, and payload. The project was developed to understand how network communication works and to gain practical experience with packet capturing and protocol analysis.

---

## Objectives

- Capture live network packets.
- Analyze packet structure and protocol information.
- Understand network communication.
- Learn how TCP, UDP, and ICMP packets work.
- Display meaningful packet information.

---

- Live packet capturing

## Installation

Clone the repository.

```bash
git clone https://github.com/YOUR_USERNAME/network-packet-analyzer-python.git
```

Move inside the project.

```bash
cd network-packet-analyzer-python
```

Install Scapy.

```bash
pip install -r requirements.txt
```

---

## Running the Program

```bash
python Packet_Capturing_Task1.py
```

Run the program as Administrator (Windows) or Root (Linux).
